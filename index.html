HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • use the <​br​> tag to insert a page break.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Functions can be reduced with arrow functions
  • Traditional anonymous function (function (a) { return a + 100; }); 1. Remove the word "function" and place arrow between the argument and opening body bracket (a) => { return a + 100; }; // 2. Remove the body braces and word "return" — the return is implied. (a) => a + 100; 3. Remove the parameter parentheses a => a + 100; Copy to Clipboard
  • Boolean statements can also be simplified
  • if (comment.id === 12483) { return true; } can be simplified to return comment.id === 12483 Note that this is exclusive to boolean if statements, with the statement following "return" being the truthy condition.
